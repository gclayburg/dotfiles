dotfiles-universal
==================

universal Linux/Unix dotfiles for the HOME directory of any user


These set of home directory dot files are known to work on a variety of unix or linux systems.  It has been tested on:
* CentOS
* RedHat
* Ubuntu
* LinuxMint

* Solaris x86
* Open Solaris

* IBM AIX
* Cygwin

* busybox
* VMware esxi 4.1
* Synology diskstation

And these shells
* bash 2.x
* bash 3.x
* bash 4.x
* ash
* sh
* ksh
* dash
